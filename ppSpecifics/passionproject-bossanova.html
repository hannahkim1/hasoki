bossa nova

an informational website

brazilian jazz, samba, smooth music that you could fall asleep to, study to, dance to
i am proud to say that i coded, researched, and source found all photos included

website link

i initally started with a more childish and colorful approach
i believed that a website with brighter colors and contrasting text would look good


i used milanote to plan out the information and general theme of the website
i used figma to mockup an image of what I want my website to look like

i then decided that the colors and overall design did not match with what I wanted to convey
i was finding a lot more research about the messaging behind bossa nova (hidden racism, activism, etc) and I decided I needed a less "childish" look

i used warmer, neutral colors
i tried staying simplistic with the design, this was my first website ever coding, so i didn't want to be too fancy

i used visual studio code to actually code
i used the languages html and css with a little bit of javascript
i utilized the animate css library to add in some mini animations

all of my research is on this milanote board!

reasoning behind this project

Hope to introduce bossa nova to some people, as well as personally go into more depth and record my findings
